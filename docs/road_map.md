# Tolmach — Road Map

## visual_context_analysis

**Статус:** idea

Скачивание видео целиком (не только аудио) с извлечением скриншотов через заданный интервал (10 / 15 / 30 секунд). Скриншоты используются для обогащения контекста при генерации отчёта.

### Суть

1. При скачивании видео — сохранять видеофайл (а не только аудио)
2. Через ffmpeg извлекать кадры с заданным интервалом (`-vf fps=1/N`)
3. Каждый скриншот сопоставлять с блоком транскрибации в окне `timestamp ± 10 секунд`
4. Отправлять в LLM пары: скриншот + соответствующий фрагмент транскрипции
5. LLM анализирует и текст, и визуальный контекст — итоговая выжимка значительно точнее
6. В финальный Markdown-отчёт встраиваются скриншоты с подписями из анализа

### Детали

- Интервал скриншотов — конфигурируемый (по умолчанию 15 секунд)
- Окно сопоставления с транскрипцией — `[screenshot_time - 10s, screenshot_time + 10s]`
- Формат скриншотов — JPEG (баланс качества и размера для LLM)
- Отчёт с картинками — Markdown с inline-изображениями (относительные пути или base64)
- Нужна мультимодальная модель (Claude Sonnet / Opus с vision)
