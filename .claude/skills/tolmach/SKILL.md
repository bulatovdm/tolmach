# Tolmach — Video Transcription & Report

Ты получаешь URL видео или путь к локальному файлу в $ARGUMENTS. Выполни все этапы последовательно.

ВАЖНО: Все временные файлы создавай ТОЛЬКО в `~/.tolmach/tmp/`. Перед началом работы создай эту директорию:
```bash
mkdir -p ~/.tolmach/tmp
```

## Этап 1: Определение источника и получение метаданных

Определи тип источника:
- **YouTube/Vimeo URL** — используй `yt-dlp`
- **Локальный файл** — проверь что файл существует

Для URL — получи метаданные и скачай аудио одной командой:
```bash
cd ~/.tolmach/tmp && yt-dlp --print title --print uploader --print duration_string "$URL"
```

Для локального файла — используй ffprobe:
```bash
ffprobe -v quiet -print_format json -show_format "$FILE"
```

## Этап 2: Скачивание аудио

Для URL:
```bash
cd ~/.tolmach/tmp && yt-dlp -x --audio-format wav --audio-quality 0 -o "%(id)s.%(ext)s" "$URL"
```

Файл будет сохранён как `~/.tolmach/tmp/VIDEO_ID.wav`.

Для локального файла — извлеки аудио через ffmpeg:
```bash
ffmpeg -i "$FILE" -vn -acodec pcm_s16le -ar 16000 -ac 1 ~/.tolmach/tmp/audio.wav
```

После скачивания проверь что WAV-файл существует:
```bash
ls -la ~/.tolmach/tmp/*.wav
```

## Этап 3: Транскрибация

Запусти whisper-cli, указывая полные пути:
```bash
whisper-cli -m ~/.tolmach/models/ggml-large-v3-turbo.bin -f ~/.tolmach/tmp/VIDEO_ID.wav --language auto --print-progress -of ~/.tolmach/tmp/transcription -otxt
```

Прочитай результат:
```bash
cat ~/.tolmach/tmp/transcription.txt
```

## Этап 4: Генерация отчёта

На основе метаданных и транскрипции создай отчёт. Определи уровень детализации по длительности видео:

### Короткое видео (до 15 минут)

Ты — аналитик видеоконтента. Проанализируй транскрипцию и создай отчёт в Markdown:

1. **Краткое содержание** (3-5 предложений) — основная суть видео
2. **Ключевые темы** — маркированный список основных тем/разделов
3. **Основные тезисы** — нумерованный список ключевых идей и выводов
4. **Цитаты** — 3-5 наиболее значимых цитат из транскрипции (дословно)
5. **Заключение** — краткий итог и основной вывод

Пиши ёмко, но информативно.

### Среднее видео (15-45 минут)

1. **Краткое содержание** (5-8 предложений) — основная суть видео
2. **Ключевые темы** — маркированный список, раскрой каждую тему 1-2 предложениями
3. **Основные тезисы** — нумерованный список (8-15 тезисов)
4. **Цитаты** — 5-8 наиболее значимых цитат из транскрипции (дословно)
5. **Заключение** — развёрнутый итог с основными выводами (3-5 предложений)

Видео средней длительности — дай достаточно подробный анализ, раскрывая каждую тему.

### Длинное видео (более 45 минут)

1. **Краткое содержание** (8-12 предложений) — основная суть видео
2. **Ключевые темы** — маркированный список, раскрой каждую тему 2-3 предложениями с подпунктами при необходимости
3. **Основные тезисы** — нумерованный список (15-25 тезисов)
4. **Цитаты** — 8-12 наиболее значимых цитат из транскрипции (дословно)
5. **Заключение** — развёрнутый итог с ключевыми выводами и рекомендациями (5-8 предложений)

Это длинное видео — создай подробный и глубокий анализ, пропорциональный объёму материала. Не упускай важные детали и нюансы.

## Формат отчёта

Отчёт на русском языке. Добавь заголовок с метаданными:

```markdown
# [Название видео]

- **Автор:** [автор]
- **Длительность:** [длительность]
- **URL:** [url или путь к файлу]
- **Дата отчёта:** [текущая дата]

---

[содержание отчёта по структуре выше]
```

Не добавляй информацию, которой нет в транскрипции.

## Этап 5: Сохранение

Сохрани отчёт в `~/.tolmach/reports/[slug-названия].md`.

## Этап 6: Очистка

Удали все временные файлы:
```bash
rm -rf ~/.tolmach/tmp/*
```

Выведи путь к сохранённому отчёту.
